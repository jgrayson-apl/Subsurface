Subsurface Visualization

This application shows how we can use the ArcGIS API for JavaScript to provide custom tools for subsurface or underground visualizations, measurements between features, and 3d distance based selections.

This application is built using:
  • Esri ArcGIS API for Javascript 4.x
  • Esri ApplicationBase  &  Configurable App Example
  • Esri Calcite Web Framework
